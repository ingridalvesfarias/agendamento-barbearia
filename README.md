# 💈 Barbearia Premium - Agendamento Online

Sistema de agendamento de luxo desenvolvido com foco em experiência do usuário (UX) e design responsivo. O projeto utiliza uma estética *Dark & Gold* para transmitir exclusividade e profissionalismo.

## 🛠️ Tecnologias Utilizadas
* **HTML5**: Estrutura semântica do formulário.
* **CSS3**: Design personalizado com foco em:
    * **Glassmorphism**: Efeito de desfoque no card.
    * **Flexbox**: Centralização perfeita em qualquer dispositivo.
    * **Responsive Design**: Ajustes automáticos para Mobile e Desktop.
* **JavaScript**: 
    * Validação de datas e horários em tempo real.
    * Integração direta com a API do WhatsApp.

## ✨ Diferenciais deste Projeto
* 🌙 **Estética Premium**: Paleta de cores baseada em Dourado (#f1c40f) sobre fundo Preto Grafite.
* 🖼️ **Layout Inteligente**: Fundo com imagem descentralizada para valorizar a arte visual sem atrapalhar a leitura.
* 📱 **Mobile-First**: O formulário se adapta perfeitamente a telas pequenas, garantindo que o cliente consiga agendar de qualquer lugar.
* 💬 **Mensagem Estruturada**: Envia para o barbeiro todos os detalhes (Serviço, Barbeiro de preferência, Data e Hora) já formatados.
